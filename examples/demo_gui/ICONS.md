# DataCat Demo GUI Icons

Please save the provided icon file to this directory:

## Required Icon File

**logo-small.png**
- Size: 64x64 or 128x128 pixels
- Use: Demo GUI favicon (browser tab icon)
- Source: The small/medium-sized icon provided

## Icon Features

The icon features:
- Cute cat mascot with data visualization elements (bars/charts)
- "DataCat" text in blue-to-purple gradient
- Transparent or white background
- Modern, friendly design suitable for data monitoring application

## Usage

This icon is referenced in:
- `demo_gui.py` - Set as favicon for the Gradio application via the `favicon_path` parameter

The icon will appear in:
- Browser tabs when the demo is running
- Browser bookmarks
- Task bar (on some operating systems)
