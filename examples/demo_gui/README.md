# DataCat Demo GUI

Interactive web-based demonstration interface built with Gradio.

## Quick Start

```bash
# Install Gradio
pip install gradio

# Run the demo
.\scripts\run-demo-gui.ps1
```

The demo will open in your browser at http://localhost:7860
