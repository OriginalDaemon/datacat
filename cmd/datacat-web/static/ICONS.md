# DataCat Icons

Please save the provided icon files to this directory with the following names:

## Required Icon Files

1. **favicon.ico** (or favicon.png)
   - Size: 32x32 or 48x48 pixels
   - Use: Browser tab favicon
   - Source: The smallest icon provided

2. **logo-small.png**
   - Size: 64x64 or 128x128 pixels
   - Use: Small logo in UI headers, mobile displays
   - Source: The second smallest icon

3. **logo.png**
   - Size: 256x256 pixels
   - Use: Main logo in web UI, documentation
   - Source: The medium-sized icon

4. **logo-large.png**
   - Size: 512x512 pixels
   - Use: High-resolution displays, documentation hero images
   - Source: The largest icon provided

## Icon Features

All icons feature:
- Cute cat mascot with data visualization elements (bars/charts)
- "DataCat" text in blue-to-purple gradient
- Transparent or white background
- Modern, friendly design suitable for data monitoring application

## Usage

These icons are referenced in:
- `templates/base.html` - Favicon and header logo
- `templates/sessions.html` - Dashboard logo
- `README.md` - Documentation branding
- Demo GUI - Application icon
