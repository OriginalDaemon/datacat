# Change Summaries

This directory contains change summaries and fix documentation.

## Naming Convention

- Files should use lowercase names with hyphens (kebab-case)
- Examples: `fixes-applied.md`, `session-timeline-updates.md`, `exception-stack-trace-fix.md`
- Include dates in filenames if needed: `2024-01-15-time-range-fix.md`

## Purpose

Change summaries document:
- Bug fixes
- Feature additions
- UI improvements
- API changes
- Breaking changes
